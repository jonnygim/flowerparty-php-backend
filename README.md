## 주요 기능📍
- 회원 로그인 기능
- 실내 정원 식물 open API 활용해서 식물 정보 제공 & 개인 식물 저장 기능
- IoT 기술을 접목한 화분의 센서 정보를 홈 화면에 출력해주는 기능 (온도, 습도, 조도)
- 블루투스를 통한 화분 급수 기능
- 식물 일지 기능
